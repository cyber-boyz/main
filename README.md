#### 1\. List out and explain the features of the four major types of raster image file extensions. Point out the image file extension that supports both transparency as well as animation.   



###### &#x20;                                                                                                                                                                                                                                                 - JPEG (Joint Photographic Experts Group)                                                                                                                                  

###### &#x20;  Compression: lossy (reduces file size by discarding some data)                                                                                                        

###### &#x20;  Best for : Photographs and web images where small size matters                                                                                                       

###### &#x20;  Limitations : No transparency, repeated editing reduces quality 

###### 

###### &#x20;                                                                                                                                                                                                                                                     - GIF ( Graphics Interchange format)                                                                                                                                   

###### &#x09;Compression: lossless, but limited to 256 colors.                                                                                                                     

###### &#x09;Best for : Simple graphics , icons and short animations                                                                                                             

###### &#x20; 	Limitations : Not suitable for high quality photos due to limited colors  

###### 

###### &#x20;                                                                                                                                                                                                                                                - PNG ( Portable network Graphics)                                                                                                                                  

###### &#x20;	Compression: lossless, (retains all image data).                                                                                                                     

###### &#x09;Best for : web graphics , logos and images needing transparency                                                                                                  

###### &#x09;Limitations : Larger file sizes compared to JPEG.                                                                                                                                             

###### 

###### &#x20;                                                                                                                             

###### \- TIFF ( Tagged Image File format)                                                                                                                                         

###### &#x09;Compression: lossless, very high quality                                                                                                                              

###### &#x09;Best for : Printing, Publishing and professional scanning                                                                                                             

###### &#x09;Limitations : Very large file sizes, not ideal for web use.

###### &#x20;                                                                                                                                                                                                                                                                            The raster image file extension that supports both transparency and animation is GIF.





##### 2\. Explain the difference between dithering and anti-aliasing.

###### &#x20;                                                                                                                                                                                                                                                                                  Dithering:



Dithering is a technique used to simulate colors or shades that are not available in a limited color palette.



It works by arranging pixels of different colors close together so that the human eye perceives them as a new blended color.



Commonly used in GIFs, low-color images, and printing to give the illusion of smooth gradients.



Example: Mixing black and white pixels to create the appearance of gray.



###### Anti-Aliasing:



Anti-aliasing is a technique used to smooth jagged edges (aliasing) in digital images and graphics.



It works by blending the edge pixels with background colors to reduce the stair-step effect.



Commonly used in text rendering, vector graphics, and 3D models to make visuals appear smoother.



Example: Smoothing the diagonal edges of letters on a screen.
